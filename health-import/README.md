# Apple Health 数据导入目录

将 iPhone 导出的 `apple_health_export.zip` 放在此文件夹，然后在平台中点击「扫描 health-import 文件夹」即可导入。

## 导出步骤（Apple Watch 用户）

1. 确保 Apple Watch 已同步到 iPhone「健康」App
2. iPhone → **健康** → 右上角头像 → **导出所有健康数据**
3. 通过 AirDrop 将 zip 文件发送到 MacBook
4. 将 zip 放入本文件夹，或在网页中直接上传

数据仅保存在本机，不会上传到任何服务器。
